##  Overview

**Client Tracker** is a cutting-edge desktop application specifically designed for Māori support organizations to track participant progress through structured programs. Built with Python  and featuring an intuitive modern interface, it provides comprehensive tools for managing participant information, tracking 26-week program phases, and monitoring long-term advocacy support extending up to **6 years** .

>  **Beautiful UI** |  **Powerful Analytics** |  **Secure Data** |  **Purpose-Built**

##  Usage Guide

###  Adding a Participant

`
1. Navigate to  Participants tab
2. Fill in required fields:
    Name, Age, Gender
    Location
    Iwi and Hapū affiliations
    Start date (YYYY-MM-DD)
    Assigned Kaimahi
    Advocacy types (if needed)
3. Click " Add Participant"
`

###  Tracking Progress

`
1. Select participant from list
2. Go to  Progress tab
3. View real-time visualization
4. Assign programs via " Set Programs"
5. Conduct weekly assessments
`

###  Conducting Assessments

`
1. Select participant in Red Phase (Weeks 1-6)
2. Click " Weekly Assessment"
3. Rate performance (0-10 scale)
4. Add detailed notes
5.  Save assessment
`

###  Generating Reports

`
1. Navigate to  Reports tab
2. View quick statistics
3. Click " Generate Statistics Report"
4. Use  Data tab for PDF export
`

##  Program Phases Explained

###  Red Phase (Weeks 1-6) - *Whitiwhiti Kōrero*
**Focus:** Initial Engagement
-  Building trust and rapport
-  Creating safe spaces for expression
-  Understanding participant needs
-  Identifying barriers to progress
-  Weekly structured assessments

###  Orange Phase (Weeks 7-14) - *Program Implementation*
**Focus:** Active Participation
-  Engagement in assigned programs
-  Skill development activities
-  Continuous progress monitoring
-  Intervention adjustments as needed

###  Green Phase (Weeks 15-26) - *Whānau Integration*
**Focus:** Community Reintegration
-  Strengthening community connections
-  Family support enhancement
-  Building independence
-  Preparing for program completion

###  Advocacy Support (Up to 6 years)
**Focus:** Long-term Specialized Support
-  Extended support for complex cases
-  Multiple advocacy types available
-  Flexible duration (6 months to 6 years)
-  Progress tracked quarterly
